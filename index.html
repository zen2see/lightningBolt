UTF-8 Encoding in hex 0xF0 0x9F 0x97 0xB2

ON LINUX TERMINAL:
printf '\xF0\x9F\x97\xB2\n' OR printf '\U1F5F2'
You can also paste (CTRL+V) the actual icon in the terminal and have hexdump show the hex values:
printf (CTRL+V) | hexdump after pasting the icon, you PIPE (|) it to the hexdump program which will display the code.

You can paste the lightning icon OR put the actual hex code before the PIPE (|) to hexdump:
printf (CTRL+V) | hexdump
printf '\xF0\x9F\x97\xB2' | hexdump
It's the same hex code for the lightning bolt 🗲:
0000000 9ff0 b297
0000004

You can printf OR echo unicode code before the PIPE (|) to hexdump:
printf '\u26A1\n' | hexdump
echo -e '\u26A1' | hexdump
It's the same hex code for the High Voltage Sign this time:
0000000 9ae2 0aa1
0000003
echo -e '\u26A1' the -e allows Various Escape characters like \u to format your argument.
\u26A1 is actually the High Voltage Sign. ⚡

URLENCODE / DECODE
urlencode [-m|-d] string [string ...]
-m Instead of full conversion, do GridSite: "mild URL encoding"
in which A-Z a-z 0-9 . = - _ @ and / are passed through unmodified.
This results in slightly more human-readable strings but the application
must be prepared to create or simulate the directories implied by any slashes.
-d Do URL decoding rather than encoding, according to RFC 1738. %HH and %hh strings are
converted and other characters are passed through unmodified, with the exception that + is converted to space.
For example, the URL http://www.gridpp.ac.uk/ becomes, http%3A%2F%2Fwww.gridpp.ac.uk%2F
urlencode & = %26
urlencode -d %26 = &

ON LINUX TERMINAL:
The format of the string for foreground color is:
"\033[ + color code + 0 or 1 (meaning normal or bold) + m"
and for background is:
"\033[ + color code + m"
A simple example of red text:
printf "\033[91mThis is red text"
An example of red text on a white background:
printf "\033[91m\033[107mThis is red text on a white background"
